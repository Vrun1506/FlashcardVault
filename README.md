This is Flashcard Vault, an AI-powered flashcard generation tool that takes you from a PDF to a set of Anki flashcards in a matter of minutes. 

Perfect for when you have to create resources for exams, and saves you so much time!

Tech stack: TypeScript, Python, Next.js, FastAPI, Firebase.
